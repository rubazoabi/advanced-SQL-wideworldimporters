# advanced-SQL-wideworldimporters
This project demonstrates advanced SQL querying techniques using the WideWorldImporters sample database.  The goal is to extract business insights from a real-world relational database using complex SQL queries and analytical logic.
## Database
- WideWorldImporters (Microsoft SQL Server sample database)
## Key SQL Topics
- Complex joins across multiple tables
- Aggregations and GROUP BY
- Window functions (RANK, ROW_NUMBER, etc.)
- Common Table Expressions (CTEs)
- Subqueries
- Conditional logic using CASE
- Ranking and performance analysis
## Project Files
- `advanced_sql_queries.sql` – collection of advanced SQL queries
## Technologies Used

- SQL Server
- WideWorldImporters database

## Project Purpose
This project demonstrates strong SQL skills, including the ability to query complex relational databases and extract meaningful business insights using advanced SQL techniques.
